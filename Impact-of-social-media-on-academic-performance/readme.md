📊 Impact Analysis Power BI Dashboard

This project showcases an Impact Analysis Dashboard built with Power BI using Excel-exported datasets. The dashboard is designed to highlight business KPIs, performance trends, and impact measures through clean and interactive visualizations.

🔑 Features

Excel data imported, cleaned, and transformed in Power Query

DAX measures for accurate calculations

Interactive filters & slicers for dynamic analysis

Elegant and professional dashboard design

🛠️ Tools Used

Power BI

Excel (data source)

DAX, Power Query

📊 Applications

Business performance monitoring

Impact tracking & KPI measurement

Data-driven decision-making
